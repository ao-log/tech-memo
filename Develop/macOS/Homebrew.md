Homebrew ではパッケージのことを formula と呼ぶようだ。

インストールすると、Cellar (```/usr/local/Cellar```) に格納される。

##### formula の追加

```
$ brew install FORMULA
```


##### formula の削除

```
$ brew remove FORMULA
```

##### formula 一覧

```
$ brew list
```
