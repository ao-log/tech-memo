# BlackBelt

[AWS Certificate Manager](https://pages.awscloud.com/rs/112-TZM-766/images/20181219_BlackBeltSeminar_AWS_Certificate_Manager.pdf)

* 機能
  * パブリック DV 証明書の発行
  * プライベート証明書の発行
  * 証明書のインポート
  * 展開 (ELB, CLoudFront, API Gateway, プライベート証明書を EC2 に)
* 証明書の更新も自動化
* Private CA
  * ユースケース: サーバー証明書、クライアント証明書など
